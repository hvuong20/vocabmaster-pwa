# VocabMaster — Project Guide

Rules được tổ chức trong `.claude/rules/`. Mỗi file có một chủ đề rõ ràng:

| File | Nội dung | Load |
|---|---|---|
| `project-overview.md` | Tech stack, file structure, deployment URLs, triết lý | Always |
| `vocabulary-data.md` | Cấu trúc topic/word, danh sách topics theo level, quy tắc thêm từ | Data files |
| `coding-standards.md` | JS/CSS patterns, naming conventions, SW cache rules, localStorage keys | JS/CSS files |
| `design-system.md` | CSS variables, typography, animations, mobile targets | CSS/HTML files |
| `pwa-notifications.md` | Push server, VAPID, SW events, NotificationScheduler, Reminder screen | Notification work |
| `development-workflow.md` | Local dev, git/deploy, checklist, common tasks, troubleshooting | On request |

**Memory (trạng thái project):** `.claude/memory.md`
